## Step 6:

Create the computerChoice() function:

```java
public static String computerChoice() {
        double computerChoice = Math.random() * 3;
        switch((int)computerChoice) {
            case 0: return "rock";
            case 1: return "paper";
            case 2: return "scissors";
            default: return "Not valid choice";
        }
```
